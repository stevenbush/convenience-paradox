# Formal research report figures (committed)

These SVG files are **copies** of the figures produced under
`data/results/campaigns/.../report_assets/formal_report_v2/figures/` for a validated offline campaign.

They are tracked in git so that [formal_research_report.md](../../../formal_research_report.md) renders correctly on GitHub, while `data/results/` remains ignored for large generated outputs.

**Refresh after regenerating the formal report:** copy the new `figure_*.svg` files from the campaign `figures/` directory into this folder (same filenames), then commit.
